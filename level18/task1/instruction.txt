1. Підготовка (with open):

Створити файл rules.txt у поточній папці.

Записати туди текст: "Не списувати! Коментувати код!".

2. Структура папок (os):

Створити головну папку Python_Course.

У ній створити 5 папок: Lesson_1, Lesson_2 … Lesson_5.

3. Наповнення (with open):

У кожній папці Lesson_X створити порожній файл main.py.

У кожній папці створити файл info.txt, де всередині написано:
"Це папка для уроку номер X".

4. Копіювання (shutil):

Скопіювати файл rules.txt (який ми створили на початку) всередину кожної папки
